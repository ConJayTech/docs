# Getting started

## Before You Begin

In order to use the API's service, the user must have the following:

* An account on the IQM platform
* At least one active Campaign
* A Client ID and Client Secret

Follow these steps to meet the above conditions:

1. [Sign Up<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-linecap="round" stroke-linejoin="round" width="1em" height="1.2em" stroke-width="1.5"><path d="M12 6h-6a2 2 0 0 0 -2 2v10a2 2 0 0 0 2 2h10a2 2 0 0 0 2 -2v-6"></path><path d="M11 13l9 -9"></path><path d="M15 4h5v5"></path></svg>](https://app.iqm.com/#/signup)  for an IQM Account:
   * Use a work email as a username and create a secure password
     * Password must be at least 8 characters in length
     * Requires at least 1 uppercase letter
     * Requires at least 1 lowercase letter
     * Requires at least 1 special character
1. To Run a Campaign:
   * See the [Create a New Campaign Guide<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-linecap="round" stroke-linejoin="round" width="1em" height="1.2em" stroke-width="1.5"><path d="M12 6h-6a2 2 0 0 0 -2 2v10a2 2 0 0 0 2 2h10a2 2 0 0 0 2 -2v-6"></path><path d="M11 13l9 -9"></path><path d="M15 4h5v5"></path></svg>](https://help.iqm.com/en/articles/5651476-create-a-new-campaign)
1. Email [integrations@iqm.com<svg xmlns="http://www.w3.org/2000/svg" width="1.25em" height="1.25em" viewBox="0 0 21 21"><g fill="none" fill-rule="evenodd" stroke="currentColor" stroke-linecap="round" stroke-linejoin="round"><path d="M3.5 6.5v8a2 2 0 0 0 2 2h10a2 2 0 0 0 2-2v-8a2 2 0 0 0-2-2h-10a2 2 0 0 0-2 2"/><path d="m5.5 7.5l5 3l5-3"/></g></svg>](mailto:integrations@iqm.com) to Request a Client ID and Client Secret.
