# Migrate a Bid Model

## Overview

The Trade Desk uses terminology similar to IQM to describe Bid Modeling.

IQM's [Bid Model API](/docs/Guidelines/Bid-Model-API-Guidelines) optimizes Campaigns for better performance. The user can fine-tune where their ads appear, manage priorities, and Campaign dimensions. The Trade Desk refers to their bid modeling as [Bid Lists<ExternalLink />](https://partner.thetradedesk.com/v3/portal/api/doc/BidList).  

The tables below highlight the main definitions, actions, and API methods for managing Bid Modeling. Each IQM endpoint is paired with the analogous The Trade Desk API. Both link to their respective documenation. By following the workflow instructions and comparing APIs, the user can migrate their Bid Modeling data from The Trade Desk to IQM.
