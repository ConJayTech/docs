# API Guidelines

Explore the common endpoints and methods associated with each of IQM's applications.

The Guidelines pages provide overviews of each application and comperehensive coverage of each endpoint supplemented with request and response samples in JSON and TypeScript. See the [TypeScript prerequisite](/docs/Getting%20Started/Typescript-Prerequisites.mdx) page for more info on how to make TypeScript API calls.
