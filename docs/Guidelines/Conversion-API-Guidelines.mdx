# Conversions API

## Overview

The Conversions API allows users to get details, lists, and create Conversions to track user traffic for advertising campaigns. 

The IQM platform supports postback and pixel type conversions. Pixel conversions are client-side tracking whereas Postback Conversions are server-side.

This page covers the common endpoints and methods associated with the Conversions API. 

Learn more about Conversions with our [Help Center articles<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-linecap="round" stroke-linejoin="round" width="1em" height="1.2em" stroke-width="1.5"><path d="M12 6h-6a2 2 0 0 0 -2 2v10a2 2 0 0 0 2 2h10a2 2 0 0 0 2 -2v-6"></path><path d="M11 13l9 -9"></path><path d="M15 4h5v5"></path></svg>](https://help.iqm.com/en/collections/3219750-choose-your-conversion).
