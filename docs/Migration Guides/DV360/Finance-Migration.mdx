# Migrate Finances

## Overview

The API Comparison table highlights the shared actions and API methods between IQM, DV360 and Campaign Manager 360 (CM360) for creating and managing Finances, Invoices, and Payments. Select an endpoint to review each platform's relevant API documentation. By comparing APIs, you can migrate your workflow from DV360 to IQM.

For a comprehensive overview of IQM's finance services, see the [Finance API](/Guidelines/Finance-Guidelines).
