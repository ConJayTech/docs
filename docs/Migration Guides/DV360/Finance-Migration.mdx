# Migrate Finances

## Overview

The API Comparison table highlights the main actions and API methods for creating and managing Finances, Invoices, and Payments. Select an endpoint from IQM, DV360 or Campaign Manager 360 (CM360) to review that platform's relevant API documentation. By comparing APIs, you can migrate your workflow from DV360 to IQM.
