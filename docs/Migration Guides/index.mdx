# Migration Guides

## Overview

Our migration guides are designed to help you transition from other platforms to IQM's API. These guides provide a side-by-side comparison of key platform entities, terminology, and functionality, making it easy to transfer your existing processes to IQM's platform. 

Each guide includes:
  * **Platform comparisons** - Understand how core entities are structured and named across different platforms compared to IQM. 
  * **API protocols and endpoints** - Explore side-by-side API actions and endpoint comparisons to ensure a smooth migration of Campaigns, Bid Modeling, Conversions, inventory management, and reports to IQM.
