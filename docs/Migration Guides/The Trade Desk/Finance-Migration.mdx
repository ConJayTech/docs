# Migrate Finances

## Overview

The API Comparison table highlights the main actions and API methods for creating and managing Finances, Invoices, and Payments. Select an endpoint from IQM or The Trade Desk to review that platform's relevant API documentation. By comparing APIs, you can migrate your workflow from The Trade Desk to IQM.
