# Migrate a Bid Model

## Overview

Beeswax uses terminology similar to IQM to describe Bid Modeling.

IQM's [Bid Model API](/docs/Guidelines/Bid-Model-API-Guidelines) optimizes Campaigns for better performance. The user can fine-tune where their ads appear, manage priorities, and Campaign dimensions. See the Beeswax [Bid Modifiers<ExternalLink />](https://api-docs.freewheel.tv/beeswax/v2.0/docs/bid-modifiers) page.  

The tables below highlight the main definitions, actions, and API methods for managing Bid Modeling. Each IQM endpoint is paired with the analogous API for Beeswax. Both link to their respective documenation. By following the workflow instructions and comparing APIs, the user can migrate their Bid Modeling data from Beeswax to IQM.
