# Migrate an Inventory

## Overview

Beeswax refers to deals in the same way as IQM. They use <var>guaranteed</var> for  Programmatic Guaranteed (PG); and <var>private_auction</var> as a Private Marketplace (PMP) Deal. This page foregrounds IQM's usage. 

IQM's [Inventory API](/docs/Guidelines/Inventory-API-Guide) manages inventories, inventory groups, and deals.

The table below highlights the main actions and API methods for managing inventory and deals. Each IQM endpoint is paired with the analogous API for Beeswax. Both link to their respective documentation. By following the workflow instructions and comparing APIs, the user can migrate their data from Beeswax to IQM.
